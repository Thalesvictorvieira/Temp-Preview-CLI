# Skymio CLI

Aplicação de terminal em Python para consultar previsão do tempo utilizando a API WeatherAPI.

## Funcionalidades

- Previsão do tempo atual
- Previsão para os próximos dias
- Exibição do dia da semana
- Interface CLI interativa

## Tecnologias

- Python
- Requests
- WeatherAPI
